## 🔬 七款产品深度剖析

### 1. Claude Code — 终端 Agent 之王

!!! success "核心优势"
    - **100 万 token 上下文**：可一次性读入整个代码库
    - **终端原生**：CLI 工具，无需 IDE，适合远程服务器
    - **Subagent 链**：自动分解复杂任务为子任务
    - **CLAUDE.md**：通过配置文件注入项目上下文

!!! warning "注意事项"
    - Pro 限流 45 次/5 小时
    - 无 IDE 内联补全
    - 价格阶梯较高

| 方案 | 价格 | 说明 |
|------|:---:|------|
| Pro | $20/月 | 基础用量 |
| Max 5x | $100/月 | 5 倍限额 |
| Max 20x | $200/月 | 20 倍限额 |

**适用场景：** 大型重构、跨文件修改、夜间无人值守 Agent

---

### 2. Cursor — IDE 日常编码首选

!!! success "核心优势"
    - **内联补全接受率 72%**：业界最高
    - **Composer 2 自研模型**：SWE-bench 73.7%
    - **Background Agent**：后台持续运行
    - 原生 IDE 体验，覆盖 80% 日常场景

!!! warning "注意事项"
    - 价格梯度过大（Pro → Pro+ → Ultra）
    - 企业版数据走美国

| 方案 | 价格 | 说明 |
|------|:---:|------|
| Pro | $20/月 | 个人入门 |
| Pro+ | $60/月 | 增强用量 |
| Ultra | $200/月 | 无限制 |

**适用场景：** 日常交互编码、快速迭代

---

### 3. GitHub Copilot — 生态原生

!!! success "核心优势"
    - **PR Review 原生集成**：自动审查 Pull Request
    - **$10 入门价**：性价比最高
    - **企业合规最友好**：SOC2、默认不训练

!!! warning "注意事项"
    - Agent 能力 56% 落后于头部
    - 2026/06 改为 AI Credits 计费

| 方案 | 价格 | 说明 |
|------|:---:|------|
| Pro | $10/月 | 个人入门 |
| Pro+ | $39/月 | PR Review |
| Max | $100/月 | 无限制 |

**适用场景：** GitHub 深度用户、企业标准化部署

---

### 4. Codex CLI — 异步云端任务

!!! success "核心优势"
    - **异步架构**：提交任务后无需等待
    - **云端沙箱**：隔离执行，不占本地资源
    - SWE-bench 85.0%，仅次于 Claude Opus 4.7

!!! warning "注意事项"
    - 纯云端执行，涉密不可用
    - 无内联补全

| 方案 | 价格 | 说明 |
|------|:---:|------|
| Plus 包含 | $20/月 | ChatGPT Plus 用户可用 |

**适用场景：** 批量修正、非敏感项目

### 5. Devin Desktop（原 Windsurf）— 多 Agent 编排

!!! success "核心优势"
    - **多 Agent Kanban 看板**：可视化任务分配
    - 云端 + 桌面双模式

!!! warning "注意事项"
    - SWE-bench 45.8% 偏低
    - 品牌更名造成混乱（Windsurf → Devin）

| 方案 | 价格 | 说明 |
|------|:---:|------|
| Desktop | $20/月 | 个人版 |
| Teams | $40/月 | 团队版 |

**适用场景：** 可视化多 Agent 管理

### 6. AWS Kiro — 规格先行开发

| 方案 | 价格 | 说明 |
|------|:---:|------|
| Pro | $20/月 | 基础版 |
| Pro+ | $40/月 | 增强版 |

**适用场景：** 重视规范的企业团队
